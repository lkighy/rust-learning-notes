# Fairings

Fairing 是 Rocket 处理结构化中间件的方法. 借助 fairing , 您的应用程序可以在请求生命周期中记录或重写有关传入请求和传出相应的信息.

## Overview

实现 [`Fairing`]() 特性的任何类型都是 都是 fairing. fairing 挂在 Rocket 的请求生命周期中, 接收时间的回调, 例如请求和响应.
